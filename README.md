# Why So Crabby

Physiological and behavioral stress response of shore crabs to changes in salinity and temperature as a proxy for growth and future population changes in response to climate change.

## Repository layout

- `README.md` — project overview and working links
- `images/` — static images used across project materials
- Root-level `.docx`, `.pdf`, and `.pptx` files — project deliverables and supporting documents

## Materials

- Tanks
- Salty water
- Water pump
- Bull kelp (fish fillet if not found)
- Hemalymph draw equipment
- Respirometry equipment
- Scale
- Knife
- Tongs
- ImageJ
